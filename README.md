# GBCamTools

A set of tools for working with Game Boy Camera™ .sav files

Save parsing was based on gbcam2png:
https://github.com/raphnet/gbcam2png

Also with the help of the Game Boy Camera™ save info here:
https://github.com/Raphael-Boichot/Inject-pictures-in-your-Game-Boy-Camera-saves
